- **緊急度**: 高
- **分類**: bug
- **状態**: 対応済み

## 対応メモ

### 原因
CloudFrontからALBへHTTP接続 → ALBのHTTPリスナーがHTTPSにリダイレクト → CloudFrontに戻る → ループ発生

### 修正
ALBのHTTPリスナー(80番ポート)のデフォルトアクションを「redirect」から「forward」に変更

```bash
aws elbv2 modify-listener \
  --listener-arn "arn:aws:elasticloadbalancing:ap-northeast-1:805673386383:listener/app/meeting-visualizer-dev-alb/43601023bb23e250/239a9531a036791b" \
  --default-actions '[{"Type":"forward","TargetGroupArn":"arn:aws:elasticloadbalancing:ap-northeast-1:805673386383:targetgroup/meeting-visualizer-dev-tg/af201bf3a39b5a3f"}]'
```

### 確認
- https://meeting-dev.aidreams-factory.com/ → 200 OK ✅
- https://meeting-dev.aidreams-factory.com/api/health → healthy ✅
