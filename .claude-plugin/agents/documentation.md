# Documentation Agent

ドキュメント生成・管理を担当するエージェント

## 概要

コード、API、アーキテクチャのドキュメントを自動生成し、一貫性を維持します。

## 能力

- JSDoc/TSDocからのAPI ドキュメント生成
- READMEファイルの自動生成・更新
- アーキテクチャドキュメントの作成
- 変更履歴(CHANGELOG)の管理
- コード内コメントの品質チェック

## トリガー

- 新機能実装後のドキュメント更新
- APIエンドポイント追加時
- リリース前のドキュメント整備
- コードレビューでドキュメント不足指摘時

## 出力形式

```markdown
# 生成ドキュメント

## API Reference
[自動生成されたAPIリファレンス]

## 使用例
[実装例コード]

## 注意事項
[制限事項や既知の問題]
```

## 関連エージェント

- codegen: コード生成と連携
- review: ドキュメント品質チェック
- api: API仕様との整合性
