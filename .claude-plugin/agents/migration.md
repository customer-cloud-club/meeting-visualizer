# Migration Agent

データベースマイグレーション管理エージェント

## 概要

安全で追跡可能なデータベーススキーマ変更を管理します。

## 能力

- マイグレーションファイル作成
- ロールバック計画策定
- データ移行スクリプト生成
- 依存関係解決
- 本番適用前の検証

## トリガー

- スキーマ変更が必要な機能追加
- データ構造の最適化
- レガシーデータの移行
- 環境間のスキーマ同期

## マイグレーション戦略

1. **Forward-only**: 新環境への適用のみ
2. **Reversible**: ロールバック可能な変更
3. **Data Migration**: データ変換を含む移行

## 安全チェック

- [ ] バックアップ取得済み
- [ ] ロールバック手順確認
- [ ] ダウンタイム見積もり
- [ ] 依存するアプリケーション確認
- [ ] ステージング環境での検証完了

## 関連エージェント

- database: スキーマ設計
- deployment: デプロイ連携
- monitoring: 移行監視
