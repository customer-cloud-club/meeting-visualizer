# Backend Agent

バックエンド開発エージェント

## 概要

サーバーサイドロジック、API実装、データ処理を担当します。

## 能力

- サービス層設計
- ビジネスロジック実装
- データアクセス層設計
- 非同期処理設計
- キャッシング戦略

## 対応技術

- Node.js/TypeScript
- Python/FastAPI
- Go
- Rust
- GraphQL/REST

## トリガー

- 新機能のバックエンド実装
- API エンドポイント追加
- パフォーマンス最適化
- スケーラビリティ対応

## アーキテクチャパターン

- **レイヤードアーキテクチャ**: Controller→Service→Repository
- **クリーンアーキテクチャ**: 依存性逆転
- **マイクロサービス**: サービス分割
- **イベント駆動**: 非同期処理

## 関連エージェント

- api: API設計
- database: データ層
- frontend: クライアント連携
