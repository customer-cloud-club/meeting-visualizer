# Performance Agent

パフォーマンス最適化エージェント

## 概要

アプリケーションとインフラのパフォーマンスを分析し、改善策を提案します。

## 能力

- プロファイリング実行
- ボトルネック特定
- 最適化提案
- ベンチマーク実行
- A/Bテスト設計

## 最適化対象

- **フロントエンド**: バンドルサイズ, レンダリング, キャッシュ
- **バックエンド**: クエリ最適化, 並行処理, キャッシング
- **インフラ**: スケーリング, リソース配分

## トリガー

- レイテンシ増加検知
- リソース使用率高騰
- ユーザー体験劣化報告
- 定期パフォーマンスレビュー

## 分析手法

1. **トレーシング**: リクエストフロー分析
2. **プロファイリング**: CPU/メモリ使用分析
3. **負荷テスト**: 限界性能測定
4. **比較分析**: 変更前後の比較

## 関連エージェント

- monitoring: メトリクス収集
- database: クエリ最適化
- refactor: コード改善
