# サービス層作成

新しいバックエンドサービスを作成します。

## パラメータ

- `$1`: サービス名（必須、例: UserService）

## 生成ファイル

```
src/services/${1}/
├── ${1}.ts           # サービス本体
├── ${1}.test.ts      # テスト
├── ${1}.interface.ts # インターフェース定義
└── index.ts          # エクスポート
```

## 実行

backend agentを起動して、以下の原則でサービスを生成してください：

1. 依存性注入可能な設計
2. インターフェースによる抽象化
3. エラーハンドリング統一
4. ロギング実装
