# API エンドポイント生成

新しいAPIエンドポイントを生成します。

## パラメータ

- `$1`: リソース名（必須、例: users）
- `$2`: エンドポイントタイプ（optional: rest, graphql）

## 生成内容

1. コントローラー/リゾルバー
2. サービス層
3. バリデーション
4. テスト
5. OpenAPI仕様

## 実行

api agentを起動して、以下の形式でエンドポイントを生成してください。

```
/api/v1/${1}
  GET    / - 一覧取得
  POST   / - 新規作成
  GET    /:id - 詳細取得
  PUT    /:id - 更新
  DELETE /:id - 削除
```
